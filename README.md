# CTA 施策学習サイト

CTA（キャリアチケット就職）記事ページ施策の勝ち/負け/引き分けをデザイン観点で学ぶためのサイト。

## 使い方

`index.html` をブラウザで開くだけで動作します（CDN不使用・オフライン対応）。

## 判定基準

**案C（チームの採用判断）**: 寄せ対応・本番採用 = 勝ち、デバイス別採用・継続中 = 引き分け、未採用 = 負け
