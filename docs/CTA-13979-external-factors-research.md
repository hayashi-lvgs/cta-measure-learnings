# CTA-13979 外部要因リサーチ（2026-07-15実施・サイト未反映）

検証期間（2026/05/28〜06/15）に重なっていた施策以外の外部要因の調査結果。
**ユーザー承認が出たら summary.md とサイト詳細モーダルに反映する**（引き継ぎ書§8参照）。

## 大前提（事実）

ABC同時ランダム配信のため、外部要因は3群に等しくかかる。**群間差（C>B>A）と勝ち判定は外部要因では説明できず、頑健**。
外部要因が影響するのは ①絶対値 ②流入構成の変化（＝効果の他時期への一般化可能性） ③EF完了率などの分母構成。

## 検証期間に重なっていた外部要因（Web調査で確認済みの事実）

1. **Googleコアアップデートと期間が直接重複**
   - 2026/05/22ロールアウト開始 → 06/02完了。検証19日間のうち前半6日間がロールアウト中
   - E-E-A-T評価の厳格化が特徴。SEO流入の量・クエリの質が期中に変わった可能性
2. **6/1面接解禁＋内定率急上昇を期間中に跨ぐ**
   - 27卒内定率: 6/1時点77.2%（5/1から+10.2pt、就職みらい研究所）
   - 「内定がない」系記事の読者は母数減・焦り急増という構成変化が期中に発生
   - MCVR +267%には「6月の焦りの追い風」が増幅要因として乗っている可能性
3. **28卒サマーインターンエントリーのピークが6月**
   - 適職診断など自己分析系への潜在層（28卒）流入は季節要因だけでも増える
   - EF完了率悪化（47.6%→32.1%）は「施策による流入変化」と「季節要因」の合算の可能性

## AI推論（仮説）

- 効果量（+267%）は6月特有の高温度環境で測ったもの。閑散期にそのまま再現するとは限らない
- コアUPDで流入クエリが変わったなら、期間前半/後半で効果の出方が違う可能性（分割集計で検証可能）
- EF完了率悪化の全てを適職診断経由に帰属させるのは過剰。季節寄与を差し引くとCの実力はもう少し良い可能性もある

## 結論

| 項目 | 評価 |
|---|---|
| 勝ち判定（Cパターン寄せ） | 揺らがない（同時比較のため頑健） |
| 効果量の他時期への外挿 | 危険。6月は解禁×コアUPD×インターンピークの特異月 |
| 推奨アクション | 寄せ後の本番数値を月次モニタリング。9〜10月に効果維持を確認。可能なら期間前半/後半で分割集計 |
